Frequently Asked Questions

Question 1: Why the name Al Dente?

I liked it. It evoked an image of an amazing chef to me, a person who does not cook very well.

Question 2: What is this?

This is a cookbook website built for a class, using Blue Ribbon's cookbook recipes. We will put them in a database using SQL and get them to work with our web pages as an exercise in building interfaces with repetitive informaiton achitecture.

Question 3: I like this recipe. How do I see the instructions?

Click on the image of the tasty food

Question 4: How do I filter things?

Click on the filter description itself on the web, or on the icon in the top left for a dropdown to appear on smaller screens
