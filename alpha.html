Website to redesign: Hi-fructose - Shrinking header

I would like to impliment a shrinking header design for the art website Hi-Fructose


Trigger: The user scrolls down.
Rules: When the user scrolls down, the header navigation shrinks, and the stripes on the header dissapear. When the user scrolls back up to the top of the page, the header expands again.
Feedback: The header shrinks/grows.
Modes: While scrolling down, the header shrinks until it reaches the smaller state. While scrolling up, when in range of top of page, header expands.
